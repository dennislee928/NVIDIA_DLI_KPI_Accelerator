# NVIDIA DLI KPI Accelerator (v1.1)

Chrome 擴充功能（Manifest V3），用於在 NVIDIA DLI 課程頁面自動掃描未完成節點並透過 API 完成標記。**專為資安／API 交互測試情境設計**。

## 🚀 v1.1 更新亮點

- **實時進度條**：在彈出視窗直接查看剩餘任務量與當前百分比。
- **操作日誌**：內建簡易 Console，顯示每個節點的處理結果（成功／失敗）。
- **隨時停止**：新增「停止流程」按鈕，可隨時中斷自動化作業。
- **狀態保持**：重新開啟彈出視窗時會自動同步目前的執行進度。
- **更強健的錯誤處理**：優化 API 請求失敗時的捕捉與提示。

## 功能概要

- 在 `learn.nvidia.com` 課程內頁讀取課程大綱 API，列出未完成之 `html`／`video` 區塊。
- 依序對各節點發送完成請求（含 CSRF），並在請求間加入 **5～12 秒隨機延遲**。
- 流程結束後提示重新整理頁面以查看結果。

## 安裝（開發模式）

1. 複製或下載本專案資料夾。
2. 開啟 Chrome，前往 `chrome://extensions/`。
3. 開啟右上「開發人員模式」。
4. 點選「載入未封裝項目」，選取本專案根目錄。

## 使用方式

1. 在瀏覽器中開啟 NVIDIA DLI **課程內頁**（網址需符合 `https://learn.nvidia.com/courses/course*`）。
2. 點選擴充功能圖示，在彈出視窗中按 **「開始自動化流程」**。
3. 觀察彈出視窗中的進度條與日誌。
4. 完成後依提示重新整理頁面。
